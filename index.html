Med List



Developed with AngularJS, Med List is an app that allows users to keep track of medications by creating med lists. Lists contain medication name, along with pertinent information about the medication such as dosage, directions for use, and special notes. User lists are stored as JSON objects in the Firebase Web API. Styled with Bootstrap and SASS.

In the spirit of ‘letting food be our medicine', the Blend app, provides users with healthy smoothie recipes that target different ailments and health concerns. Users will be able to view recipes by category/concern, save their favorite recipes, and submit their own smoothie creations. Front-end built with AngularJS, utilizing token-based authentication. Backend-end API built with Ruby on Rails and SQLite. Styled with Bootstrap and SASS.
